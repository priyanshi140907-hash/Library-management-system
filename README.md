# Library-management-system
Library Management System

Introduction:
Library Management System is a simple program used to manage books in a library. With the help of this system, we can add books, issue books, return books and also search books easily.
In this project, the C++ programming language is used. Some important data structures such as Linked List, Queue and Stack are also used. Linked List is used to store book details, Queue is used for the waiting list and Stack is used to track recently returned books.

Objective:
The main objective of this project is to create a simple Library Management System that can manage books easily.
Objectives:
•	To store and manage books
•	To understand the practical use of Data Structures
•	To perform operations like Add, Issue, Return and Search books
•	To understand the use of Linked List, Queue and Stack

System Design / Algorithm:
1.	Start the program.
2.	Display menu options.
3.	User selects a choice.
4.	Perform operation:
•	Add book
•	Issue book
•	Return book
•	Search book
•	Display books
5.	Repeat until user selects Exit.
6.	Stop the program.
   
Implementation:
This project is implemented using the C++ programming language. Structure is used to store book details. Linked List is used to store books. Queue is used for the waiting list and Stack is used to track returned books. The program is menu driven so that the user can easily perform different operations.

Conclusion:
From this project we understand how data structures are used in real life applications. With the help of Library Management System, books can be managed easily. In this project, the practical use of Linked List, Queue and Stack is understood.
